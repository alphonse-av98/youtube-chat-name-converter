# YouTube Chat Name Converter

YouTubeのライブチャット欄に表示されるハンドルネーム（例: @user-id）を、元のチャンネル名に自動変換するChrome拡張機能です。

## 機能

*   **自動変換**: ライブチャット、Super Chat、メンバーシップ加入メッセージのハンドルネームをチャンネル名に置き換えます。
*   **ツールチップ表示**: 変換後の名前にマウスカーソルを合わせると、元のハンドルネームを確認できます。
*   **キャッシュ機能**: 一度取得したチャンネル名はローカルに保存され、次回以降は高速に表示されます（有効期限: 30日）。
*   **チャンネル制限**: 特定の配信者のチャット欄でのみ機能を有効にする設定が可能です。

## インストール方法

この拡張機能はChromeウェブストアには公開されていません。以下の手順で手動インストールしてください。

1.  このリポジトリをZIPでダウンロードして解凍するか、`git clone` します。
2.  Google Chromeを開き、アドレスバーに `chrome://extensions` と入力します。
3.  右上の **「デベロッパー モード」** スイッチをオンにします。
4.  左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5.  解凍したフォルダ（`manifest.json` が含まれているフォルダ）を選択します。

## 使い方

### 基本的な使い方
拡張機能をインストールした状態でYouTubeのライブ配信ページ（またはアーカイブ）を開くと、チャット欄のハンドルネームが自動的にチャンネル名に変換されます。

### 設定（オプション）
拡張機能のアイコンを右クリックして「オプション」を選択すると、設定画面が開きます。

*   **キャッシュのクリア**: 保存されているチャンネル名データをすべて削除します。
*   **対象チャンネルの制限**:
    *   特定の配信者のチャットでのみ変換を行いたい場合、その配信者のハンドルネーム（例: `@youtube`）をリストに追加します。
    *   リストが空の場合は、すべてのチャンネルで有効になります。

## 注意事項

*   YouTubeの仕様変更により、予告なく動作しなくなる可能性があります。
*   初回表示時はチャンネル名を取得するために通信が発生するため、変換まで一瞬ラグがある場合があります。
